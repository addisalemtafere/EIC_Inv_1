Responding to “IP Expansion” service request

When an authorized EIC Officer from Registration and Licensing department logs in, the system presents him with a list of customer service requests that are pending attention from EIC side. The Officer follows the following steps to respond to online customer service requests.

  1. The officer selects the  button from the left navigation panel.
  2. The officer searches a particular service request using different searching criteria such as TIN and Customer name. He can also simply select a service request pick from the available list.
  3. The officer hits the search button.
  4. The system displays results in a grid list that displays the requested service name, service status (drafted, submitted, approved, etc), Investor name, Project Name and action buttons.
  5. The officer selects the desired record and reviews the completeness of the application form completed by the customer.
  6. By clicking the Go button where the IP expansion request exists, he reviews the data provided in each tab and also opens and reviews each document attachment.

Sending Response to the Customer

After an authorized officer has reviewed data completed in each step and reviews all attached documents:

If any incomplete item is identified a notification message is sent to the investor indicating the incomplete items. The customer receives the notification in the “Investor Portal” and re-submits the application form by completing the missing items.

If data is complete and all requirements are fulfilled, the Officer sends a notification informing the customer to settle service fee payment and pick “Investment License Expansion” certificate.   

 

Settling Service Fee Payment and Certificate Generation

The system uses data defined in the “Service Fee Settings” to calculate the required payment amount for the selected service, and generate an electronic receipt using the steps presented below.

  1. Officer select payment step (tab)
  2. Officer enters the receipt number that is available on the blank perforated receipt.
  3. Officer clicks the  button
  4. System generates the receipt on the screen
  5. Officer prints the receipt and issues one copy to the customer 

Generating Certificate

After the customer has settled the service fee, the next step is generating “Commercial Registration Certificate” using the following steps.

  1. Officer selects the next step (tab).
  2. Officer clicks “Print” button
  3. System generates the required certificate
