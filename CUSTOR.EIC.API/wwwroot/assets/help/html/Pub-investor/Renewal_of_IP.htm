Renewal of Investment Permit

This section discusses the steps required for processing renewal of the investment permit. The process commences when an Investor selects the “Renewal of Investment Permit” option from the list of services available on the Investor Dashboard. A Project Renewal form appears with multiple tabs.  The following steps are used for renewal of the investment permit.

Step 1. Service Info

  1. Select “Service info” tab from the tabs
  2.  General information and the service pre-requisite for renewal of the investment permit

Step 2. Project Renewal Detail

  1. Select the “Project Renewal” tab
  2. Select the project for which the “Renewal” is processed from the project list dropdown
  3. Click “Save” and the system will redirect to the next step

 

 

Step 3. Document Attachment

In this step, the system presents the list of documents that need to be uploaded for the selected service based on the rules defied in the “Settings” module of the system.

  1. Investor clicks “Attach” button located to the right of each required document. 
  2. System displays the filename of the attached document.
  3. The above steps are repeated for each document requirement

 

 

Step 4. Confirmation

This is the last step that allows the Investor to submit the application so that an authorized EIC Officer can review and make the necessary decision.

Click “Finish” button to submit the application to be processed by designated EIC staff.
