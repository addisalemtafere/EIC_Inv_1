Processing Duty-free Incentive Requests

Step 1: Select “Duty Free” from the list of services

Step 2: Select the project for which the “Duty Free Incentive” is requested

 

 

Step 3: Press “Continue”. The system displays the list of documents that need to be uploaded.  

 

Step 4: Attach all necessary documents and press “Submit” button. The system will move to the next step – “Fill Request Form”.

 Step 5: Fill in the data items provided on the “Request Form” as shown below.

  1. Select customs branch
  2. Select incentive category
  3. Enter Item Quantity
  4. Select Requested Date
  5. Select Currency type
  6. Enter amount
  7. The system display currency rate from “Settings”
  8. Enter Invoice number and finally Press “Add” button to save data

 

 

Responding to “Uploading Construction Materials”

 

When an authorized EIC Officer from Incentive department logs in the system presents him/her with a list of Investor service request that are pending attention from EIC side. The Officer follows the following steps to respond to online investor service requests.

  1. The officer selects the Dashboard button from the Left navigation panel
  2. The officer searches a particular service request using different searching criteria such as TIN and Investor name. She or he can also simply select a service request pick from the service available list
  3. The officer hits the search button
  4. The system displays results in grid list that displays the requested service name, service status (drafted, submitted, approved etc.),Investor name ,project name action buttons
  5. The officer selects the desire record and reviews the completeness of the application form completed by Investor
  6. By clicking the Go button, he reviews the data provided in each tab and also opens and reviews each document attachment

The above steps are presented in the screenshot below along a marker the indicates numbered step

 

 Sending Response to the Investor

After the authorized officer has reviewed data completed in each step and reviews all attachment document

  1. If any incomplete item is identified a notification message is sent to the Investor indicating the incomplete items. The Investor Receives the notification in “Investor Portal” and re-submit the application form by completing the missing items
  2. If data is complete and all requirements are fulfilled, the Officer sends a notification informing the Investor to pick “Incentive letter”
