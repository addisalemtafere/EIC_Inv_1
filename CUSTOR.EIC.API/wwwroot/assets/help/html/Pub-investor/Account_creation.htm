Online Account Creation

Every customer that would like to use EIC’s Online Portal is required to create an online account that allows it access the different services of the portal. The steps required for creating an online account are presented below.

  1. Click “Register” button that is located on top right corner of the portal.
  2. The following dialog appears.
  3. Fill the required data for the following items.

Data Item

Description

Full Name

The Full name of the customer

Email

The Email address of the customer, which is used for sending account confirmation message and different service related messages.

Phone Number

Phone number of the customer

User name

This is used as User Id that is used to login to the system

Password

Unique password that is should be confirmed correctly

 

  1. Click “Save” button. The system sends account confirmation message to the email address provided by the customer.
  2. The user will open the message using her email application and click “Confirm Your Account” link. The account is actvited and the following confirmtion message is displayed.

 

Notes

  1. The system requires unique “Email” and “User Name”. An error message is displayed if either of them is already taken up.
