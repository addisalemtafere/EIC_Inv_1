Ethiopian Investment Commission’s Online System, known as eInvest, that is developed to serve the commission’s customers. The system provides the following sub-systems. Throughout this online help, follow each steps for a specified function. 

  • Service Request Receipt Notification
  • Review of Requested Service
  • Notification of missing items
  • Notification of service process completion
  • Payment Order Calculation and Fee Collection
  • Certificate Generation
  • Online Custom Duty Exception Service responding process
  • Incentive Management
