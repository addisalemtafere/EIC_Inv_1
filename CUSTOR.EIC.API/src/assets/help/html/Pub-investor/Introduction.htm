Ethiopian Investment Commission’s Online System, known as eInvest, that is developed to serve the commission’s customers. The system provides the following functionalities. Throughout this online help, follow each steps for a specified function. 

  • Requesting new, renewal, cancellation, expanssion, and substitution of investment permit.
  • Notification of missing items
  • Notification of service process status
  • Online Custom Duty Exception Service request
  • Incentive Management
