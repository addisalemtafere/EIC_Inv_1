Investment Permit Cancellation

To request for cancellation of Investment permit, the investor selects the “Cancellation of Investment Permit” item from the list of services and follow the following steps.  

Step 1. Select the Project to be Cancelled

When the investor selects the “Cancellation of Investment Permit” item, the system presents a list of projects that belong to the Investor, as show below.

Click “Select” the desired project, to start the cancellation process.

Step 2. Project Cancelation Request

  1. Make sure that you are on “Project Cancellation” tab
  2. Enter a brief description of the reason for cancellation
  3. Click “Save” button to store the data
