Managing Aftercare Data

Users from “Aftercare” department can periodically maintain data that reflect the current status of a project, including:

  • Project Cost
  • Employment Details
  • Data on Utilities
  • Project Outputs
  • Project Input
  • Project Status and Stage

This section discusses the steps required for maintaining follow up data.   

Step 1: Select the Investor

The Aftercare officer starts the service request by selecting “Aftercare” from the list of menu items from the left navigation provided on the Aftercare Dashboard. The system presents a list of investors.

  1. Enter the searching criteria like TIN
  2. Hit the “Search” button   
  3. The list of investors that meet the search criteria are displayed.
  4. Select the investor for which the aftercare data is maintained 

Step 2: Select Project

From the list of projects, the aftercare officer selects the desired project.  

Step 3: Follow up

In this step, the aftercare officer is provided with the list of aftercare data groups as shown below..

Project cost

From the given list, click on the “Project cost”. The system displays the planned project cost data entered during “New Project Permit” process in yellow. Then click “Add” button to add the project cost data for the current follow-up period..

 

 The steps required for maintaining periodic follow-up data for the different data groups are similar to the steps discussed above.
